## ✨ Sobre o Projeto

Este é o meu portfólio pessoal — uma vitrine viva do meu trabalho como **Engenheiro de Software**. O projeto reúne cases reais em produção (web, mobile e plataformas com IA), habilidades técnicas mapeadas por nível e área, formação acadêmica, certificações e conteúdo de blog.

A ideia central é simples: **mostrar o que eu faço mostrando como eu faço**. Por isso o próprio site foi construído com as mesmas ferramentas e padrões que uso no dia a dia em clientes reais — do design system à arquitetura de componentes.

## 📁 Estrutura do Projeto

```bash
Meu-Portifolio/
├─ app/                     # Rotas (Next.js App Router)
│  ├─ about/                # Sobre
│  ├─ blog/                 # Blog
│  ├─ certificates/         # Certificados
│  ├─ contact/              # Contato
│  ├─ education/            # Formação
│  ├─ experience/           # Experiência
│  ├─ projects/             # Projetos
│  ├─ skills/               # Habilidades
│  ├─ globals.css           # Estilos globais
│  ├─ layout.tsx            # Layout raiz
│  └─ page.tsx              # Página inicial
├─ components/
│  ├─ home/                 # Seções da home
│  ├─ layout/               # Navbar, Footer
│  ├─ ui/                   # Design system (shadcn/ui)
│  └─ theme-provider.tsx
├─ hooks/
│  └─ use-toast.ts
├─ lib/
│  ├─ constants.ts          # Dados do portfólio (projetos, skills, etc.)
│  ├─ motion.ts             # Presets de animação
│  └─ utils.ts
├─ public/                  # Assets estáticos (cv.pdf, sw.js)
├─ assets/                  # Imagens dos projetos
├─ next.config.js
├─ tailwind.config.ts
└─ tsconfig.json
```

## ⚡ Como Rodar Localmente

### **Pré-requisitos**

![Node.js](https://img.shields.io/badge/Node.js-18+-339933?style=flat-square&logo=node.js&logoColor=white)
![npm](https://img.shields.io/badge/npm-9+-CB3837?style=flat-square&logo=npm&logoColor=white)
![Git](https://img.shields.io/badge/Git-Latest-F05032?style=flat-square&logo=git&logoColor=white)

### **Instalação**

```bash
# 1. Clone o repositório
git clone https://github.com/GuilhermeMeneguellis/Meu-Portifolio.git

# 2. Acesse a pasta do projeto
cd Meu-Portifolio

# 3. Instale as dependências
npm install

# 4. Rode o servidor de desenvolvimento
npm run dev

# 5. Abra no navegador
# http://localhost:3000
```

## 🎨 Personalização

Todos os dados do portfólio (projetos, experiências, skills, formação, certificados e blog) estão centralizados em [lib/constants.ts](lib/constants.ts). Basta editar esse arquivo para atualizar o conteúdo — sem precisar mexer nos componentes.

| Arquivo | O que personalizar |
|:---|:---|
| [lib/constants.ts](lib/constants.ts) | Dados pessoais, projetos, skills, experiências |
| [tailwind.config.ts](tailwind.config.ts) | Cores, temas e tokens de design |
| [app/globals.css](app/globals.css) | Estilos globais e variáveis CSS |
| [public/](public/) | Imagens, currículo e assets estáticos |
